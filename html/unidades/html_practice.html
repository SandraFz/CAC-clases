    Formularios

    <form>
    es la etiqueta dentro de la cual definiremos los elementos y funciones del un formulario.
    <form action="archivo.java" method="post"></form>
    action
    atributo que indica qué acción realizará el formulario.
    method
    a través de qué método se realizará la acción. Pueden ser get o post.
    enctype
    indica la forma en la que viajará la información; por ejemplo: text/plain.
    <input>
    etiqueta que recibe dato o instrucción, según se indique en su atributo type.
    type
    uno de los dos atributos más importantes y fundamentales para el correcto funcionamiento del formulario e indica el tipo de dato que debe introducirse en el input. Algunos de sus valores pueden ser:
    • text: <input type="text" name="nombre" size="15" maxlength="20" value="Sandra">
    • password: type="password"
    name
    el otro de los atributos fundamentales y se trata de asignarle un nombre al input para identificarlo más adelante.
    size
    atributo que indica la cantidad de caracteres visibles.
    maxlength
    atributo que indica la extensión real del input, contado en caracteres. O sea, la cantidad de caracteres que puede recibir el input.
    value
    atributo que permite que el input tenga un valor predeterminado. Útil para hacer más rápido el llenado del formulario.
    <textarea>
    etiqueta input que permite ingresar textos largos, a diferencia del tipo text.
    <textarea rows="4" cols="30" name="mensaje">El valor predeterminado se escribe aquí</textarea>

    Atributos:
    • rows: define el número de líneas del campo texto.
    • cols: define el número de columnas del campo.
    • name: igual que el atributo que el input tipo text.
    Lista de opciones
    tipo de menús desplegables que permite seleccionar un valor de una serie de valores predeterminados.
    <select name="estación">
    <option>Primavera</option>
    <option>Verano</option>
    <option>Otoño</option>
    <option>Invierno</option>
    </select>
    Atributos:
    • size: número de valores mostrados a la vez. Si no asignamos valor a este atributo, cuando se haga click se veremos las opciones en foma de menú desplegable
      size="2"

      size=""
    • multiple: permite la selección de varios elementos a la vez.
      <select name="estación" size="2" multiple>
    • <option>: dentro se indica cada elemento de la lista.
    • selected: muestra una opción elegida por defecto:
      <option selected>Otoño</option>
    • value: define la opción que será enviado al programa de procesamiento en caso de que el usuario elija esa opción. Aquí se puede asociar la opción a un valor más manejable como un número o carácter.
      <option value:"1">Primavera</option>
    • type="radio": aporta el elemento clickeable y para eso reemplazamos la etiqueta option por input. Por lo demás, puede tener los mismos atributos que una lista de opciones normal.
      Por ejemplo: <input type="radio" name="estacion" value="3" checked>Otoño</input>
