Ensayos HTML

Estructura básica:

<html>

<head>

<title>Mi documento básico</title>

</head>

<body>

<p>Este es el cuerpo de mi primera página HTML<p>

<p>Este segundo párrafo también forma parte del documento</p>

</body>

</html>

Etiquetas de texto

  • Títulos: <h1>, <h2>, <h3>, <h4>, <h5>, <h6.>
  • Párrafo: <p>
  • Supraíndices y subíndices: <sup> y <sub>
  • Estilos:
  • Negrita: <b>
  • Itálica: <i>
  • Colores: una de las maneres es el método hexadecimal:

    <font color="#FF0000">Rojo</font>

Etiquetas de enlace

    Tipos de enlace según el atributo href:

  • Absoluto, se escribe la url completa: Typiclub
  • Relativo, depende de la ubicación del archivo con relación al enlace que lo señala: Un pdf sobre html
  • Ancla, se usa para señalar un elemento interno a la página:

    Tipos de enlaces según el origen:

  • Internos(similar a como se llama el modal en Bootstrap, pero con anclas de ambos lados: href=#nombre en el que llama, y name=nombre en el llamado): margenes
    Se puede, incluso, anclar a un elemento específico de otra página del mismo sitio de la siguiente manera: <a href="arcivo.html#seccion">
  • Locales
  • Remotos
  • Emails (se agrega el prefijo "mailto:"): <a href="mailto:juan@gmail.com">
    Se puede agregar argumentos como contacto y otros destinatarios con método get: <a href="mailto:juang@gmail.com?subject=contacto por mail&cc=roberto@gmail.com">
  • A archivos

Etiquetas contenedores

  • <a>
  • <img>

  • src (ubicación o url de la imagen)
  • alg (descripción del logo o imagen)
  • height y width (definen altura y anchura, respectivamente.
  • Listas:

    <li> (item de lista)

    Etiquetas semánticas

    Atributos

      Atributos del body:
    • bgcolor: para especificar el color del fondo de la página.
    • background="fondo.jpg": para agregar una imagen de fondo.
    • text: para poner un color al texto dentro.
    • Estilo para enlaces:
    • link(enlaces sin visitar):Codo a Codo
    • vlink(enlaces visitados):Mi presentación
    • alink(enlaces activos):Argentina Programa
    • Estilos de márgenes:
    • leftmargin
    • topmargin
    • marginwidht
    • marginheight
