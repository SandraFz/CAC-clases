CSS

Definición: Cascading Style Sheets (hojas de estilo en cascada). Es un lenguaje que consiste en una serie de elementos mediante los cuales se declaran los estilos, y permite ahorrar código ya que los estilos se pueden definir en una sola declaración.

Usos de CSS (niveles de su uso) del más específico al menos específico.

Estilo definido para una etiqueta

<p>Esto es un párrafo en varias palabras <span style="color:green">en color verde<&span>; resulta muy fácil<&p>

Esto es un párrafo en varias palabras en color verde resulta muy fácil.

Estilo definido para una parte de la página

Con el uso de la etiqueta div se puede englobar partes de un documento para asignarle algunos estilos como el posicionamiento, color, borde, tamaño de letra, etc..

<div style="color:#000099; font-weinght:bold">
